# 常见问题与修正方法

## 找不到文件

确认文件已经复制到智能体显示的工作目录，或通过对话框重新附加文件。文件格式应为 `.xlsx` 或 `.csv`。

## 发现多个同类文件

智能体不能擅自选择。用户应明确指定需要使用的文件，或暂时移走本次任务不使用的同类文件。

## 缺少必要列

可能原因包括：选择了错误文件、表头名称发生变化、列名配置仍对应另一家药店。应检查输入文件表头，并修改列名配置中的“用户录入列名”。

## CSV 无法读取

程序支持常见的 UTF-8、UTF-8 BOM 和 GB18030 编码。如果仍无法读取，可用 Excel 打开并另存为 UTF-8 CSV。

## 无法写入结果

常见原因是同名结果文件正在被 Excel 打开，或目录没有写入权限。关闭相关文件后重试。

## DeepSeek 密钥错误

确认 API Key 已完整填写、账户余额正常且电脑可以访问互联网。验证成功并保存过的密钥会从本地 JSON 重新读取；没有保存或验证失败时需要重新输入。
